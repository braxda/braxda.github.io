College Football Recruiting Analysis

This project explores college football recruiting trends by analyzing player data using Python, SQL, QGIS, and Tableau. The goal was to examine player hometowns, team distribution, and recruiting strength through various data visualization techniques.

Project Overview

To break down recruiting trends at the highest level of college football, I:

  • Extracted player data using the CFBD API
  • Geocoded player hometowns using Geopy
  • Mapped recruiting footprints<,strong> in QGIS
  • Created Tableau dashboards<,strong> to visualize team comparisons
  • Version control in GitHub

Technologies Used

Python & SQL

Used Python scripts to pull data from the CFBD API and clean it for analysis. SQL was used to store, query, and manipulate data efficiently in PostgreSQL.

QGIS

Geocoded player hometowns and created spatial layers to analyze which regions produce the most talent.

Tableau

Built interactive dashboards to compare recruiting strengths across teams.

GitHub

Used GitHub for version control and to document SQL queries & scripts.
