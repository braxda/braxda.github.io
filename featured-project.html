College Football Recruiting Analysis

Image Source: CBS Sports

This project explores college football recruiting trends using Python, SQL, ArcGIS, QGIS, and Tableau to analyze player hometowns, team distribution, and recruiting strength.

Data Collection & SQL Queries

Data was collected using the CFBD API and processed with Python. SQL was used for storing, querying, and analyzing key insights.

Recruiting Footprint Maps

Using QGIS, I geocoded player hometowns and created spatial layers to analyze which regions produce the most talent.

These maps highlight key recruiting hotspots and how different schools dominate certain areas.
