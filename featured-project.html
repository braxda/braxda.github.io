College Football Recruiting Analysis

This project explores college football recruiting trends using Python, SQL, QGIS, and Tableau to analyze player hometowns, team distribution, and recruiting strength.

Data Collection & SQL Queries

Data was collected using the CFBD API and processed with Python. SQL was used for storing, querying, and analyzing key insights.
